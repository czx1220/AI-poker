# AI-poker


## 环境配置

conda create -n AIpoker python==3.11

conda activate AIpoker

pip install -r requirements.txt

## GIT使用
git clone git@github.com:czx1220/AI-poker.git

git pull origin main

git add .

git commit -m "update"

git push origin main

### 建立自己的分支
git branch czx

### 切换到自己的分支
git checkout czx
